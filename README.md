# Путеводитель по Башкортостану 🏔️

Интерактивный путеводитель по достопримечательностям Республики Башкортостан. Проект представляет собой современную веб-страницу, где пользователи могут изучать уникальные природные и культурные объекты региона.

## ✨ Особенности

- **Интерактивная фильтрация**: Удобная сортировка мест по категориям (Природа, Культура, Города и т.д.).
- **Галерея Lightbox**: Возможность просмотра фотографий в высоком качестве в полноэкранном режиме.
- **Адаптивный дизайн**: Сайт корректно отображается на мобильных устройствах, планшетах и компьютерах.
- **Доступность (Accessibility)**: Поддержка навигации с клавиатуры и корректная работа с экранными дикторами (ARIA).
- **Производительность**: Минималистичный код на чистом JavaScript без тяжелых зависимостей.

## 🛠 Технологии

- **HTML5**: Семантическая разметка для лучшего SEO и доступности.
- **CSS3**: Современные методы стилизации (Flexbox, Grid, CSS Variables).
- **JavaScript (Vanilla JS)**: Вся интерактивность реализована на чистом языке без использования фреймворков.

## 🚀 Как запустить

Так как это статический проект, вам достаточно просто открыть файл `index.html` в любом браузере.

Для разработки рекомендуется использовать локальный сервер, например, расширение **Live Server** в VS Code или аналогичные инструменты в Zed.

## 📁 Структура проекта

- `index.html` — основная разметка страницы.
- `styles.css` — стили оформления и анимации.
- `script.js` — логика фильтрации и работы галереи.
